# lara-vue-router

A simple Tasks app: ([demo](https://youtu.be/xBNew5b2duA))

- Laravel 5.5
- Vue 2.1
- Vue-router 3.0.1
- Axios

## Installation
Clone this repository
> ``` git clone https://github.com/dcyar/lara-vue-router.git ```

Open folder and run
> ``` composer install ```

Create database, config your **.env** file and run
> ``` php artisan migrate ```

Enjoy!
